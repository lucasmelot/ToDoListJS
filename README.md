# ✅ Lista de Tarefas (ToDo List)

Projeto desenvolvido com foco na prática de **HTML e JavaScript**, criando uma aplicação simples e funcional para gerenciamento de tarefas do dia a dia.

---

## 🚀 Objetivo

O objetivo deste projeto é aplicar conceitos fundamentais do desenvolvimento web, criando uma aplicação interativa capaz de:

* Adicionar novas tarefas
* Marcar tarefas como concluídas
* Remover tarefas da lista
* Atualizar a interface em tempo real

---

## 🛠️ Tecnologias utilizadas

* HTML5
* JavaScript

---

## 📁 Estrutura de pastas

```
📦 ToDoListJS
 ┣ 📜 index.html
 ┣ 📜 README.md
 ┗ 📜 script.js
```

---

## 🎯 Funcionalidades

* Adicionar nova tarefa
* Listar tarefas na tela
* Marcar tarefa como concluída
* Remover tarefa
* Validação de input vazio
* Contador de tarefas (opcional)
* Armazenamento no navegador com LocalStorage (opcional, mas recomendado)

---

## 🧠 Lógica aplicada

O funcionamento básico segue este fluxo:

```
1. Usuário digita uma tarefa
2. Clica no botão "Adicionar"
3. A tarefa é inserida no DOM
4. Pode ser marcada como concluída ou removida
```

## 🔧 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/todo-list.git
```

2. Acesse a pasta:

```bash
cd todo-list
```

3. Abra o arquivo `index.html` no navegador.
